# homebridge-dht

## Installation

1. Install [BCM2835](http://www.airspayce.com/mikem/bcm2835/)
2. Type `npm install homebridge-dht-plugin` in homebridge directory
3. Done!

## Config Example

```json
{
  "platform": "DhtHomebridgePlatform",
  "name": "DhtHomebridgePlatForm",
  "type": 22,
  "pin": 1,
  "roomName": "My Room"
}
```
